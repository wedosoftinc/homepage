### 기업에서 원격 근무를 도입할 때 가장 중요한 요소 중 하나가 **보안**입니다. Splashtop은 기업 및 개인 사용자를 위한 **강력한 보안 기능**을 제공하여 원격 접속 중 발생할 수 있는 보안 위협을 효과적으로 차단합니다. 특히 **Splashtop Enterprise**는 기업 환경에 최적화된 보안 옵션을 제공하여 IT 관리자가 체계적으로 원격 근무 환경을 보호할 수 있도록 돕습니다.

###  **1. 강력한 데이터 암호화**

Splashtop은 **AES-256 비트 암호화**를 적용하여 원격 접속 중 전송되는 모든 데이터를 보호합니다.

 	- AES-256은 금융기관 및 정부 기관에서도 사용하는 최고 수준의 암호화 방식

 	- 해킹 및 중간자 공격(Man-in-the-Middle Attack) 방지

 	- 원격 접속 중 데이터를 가로채더라도 해독이 불가능

###  **2. 다단계 인증(MFA, Multi-Factor Authentication)**

Splashtop은 계정 보안을 강화하기 위해 **2단계 인증(2FA, Two-Factor Authentication)**을 기본 제공하며, 기업 고객은 **SSO(Single Sign-On)** 기능도 사용할 수 있습니다.

 	- **2단계 인증(2FA):** 비밀번호 입력 후 추가 인증(OTP, 이메일 인증 등)을 거쳐 보안 강화

 	- **SSO(Single Sign-On):** Google Workspace, Azure AD, Okta 등의 인증 시스템과 연동하여 하나의 로그인으로 안전하게 접속 가능

 	- **디바이스 인증:** 승인된 장치에서만 원격 접속 가능하도록 제한

###  **3. 원격 세션 보안 및 사용자 활동 모니터링**

IT 관리자는 Splashtop을 통해 원격 접속 세션을 모니터링하고, 보안 정책을 적용할 수 있습니다.

 	- **세션 기록 및 감사 로그:** 원격 접속 기록을 저장하여 보안 위반 시 신속한 대응 가능

 	- **실시간 세션 모니터링:** IT 관리자가 원격 세션을 감시하고 필요 시 강제 종료 가능

 	- **세션 녹화 기능:** 특정 사용자의 원격 세션을 녹화하여 보안 감사 및 내부 교육 자료로 활용

###  **4. 원격 접속 권한 관리**

Splashtop은 사용자 및 디바이스 별로 원격 접속 권한을 세부적으로 설정할 수 있어, **불필요한 접근을 차단**하고 보안을 강화할 수 있습니다.

 	- **IP 제한:** 특정 네트워크에서만 접속 가능하도록 설정

 	- **접속 시간 제한:** 업무 시간 외 원격 접속을 차단하여 보안 유지

 	- **역할 기반 접근 제어(RBAC, Role-Based Access Control):** 부서별로 원격 접속 권한을 차등 부여

###  **5. 엔드포인트 보안 통합 지원**

Splashtop은 다양한 보안 솔루션과 통합하여 기업의 **엔드포인트 보안**을 더욱 강화할 수 있습니다.

 	- **안티바이러스 및 EDR(Endpoint Detection and Response) 통합** 가능

 	- **기업 방화벽 및 VPN과 연동 가능**하여 내부 보안 환경 유지

 	- **자동 로그아웃 및 세션 타임아웃** 설정으로 불필요한 접속 차단

###  **6. 무비밀번호 로그인 및 보안 설정 자동화**

Splashtop Enterprise는 보안성을 유지하면서도 **편리한 로그인 방식**을 제공합니다.

 	- **SSO 로그인 지원**으로 복잡한 비밀번호 없이 안전하게 접속 가능

 	- **비밀번호 없는 원격 로그인**(Passwordless Authentication) 지원

 	- **보안 정책 자동 적용:** IT 관리자가 기업 전체에 동일한 보안 정책을 배포 가능

###  **Splashtop 보안 기능이 제공하는 가치**

 **기업의 기밀 데이터 보호** – 원격 접속 중 데이터 유출 방지
 **해킹 및 무단 접속 차단** – 강력한 인증 시스템으로 보안 강화
 **IT 부서의 관리 효율성 향상** – 중앙 관리 기능으로 보안 정책 일괄 적용
 **원격 근무의 안정성 확보** – 안전한 원격 환경에서 업무 가능

### 

Splashtop은 **최고 수준의 보안 기술**을 적용하여, 기업과 개인 사용자가 안심하고 원격 근무를 할 수 있도록 돕습니다. 강력한 암호화, 다단계 인증, 세션 모니터링, 접속 권한 관리 등의 기능을 활용하면 **보안 위협 없이도 편리하게 원격 업무를 수행**할 수 있습니다.
