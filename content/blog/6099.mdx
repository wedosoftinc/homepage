# [
![](/blog/images/post-6099-1759372074303-i0dwa5.png)
](http://wedosoft0527.mycafe24.com/wp-content/uploads/2023/05/eSignature-in-CPQ.png)

 	- CPQ 부가 기능에 추가 비용 없이 Dropbox Sign으로 구동되는 전자 서명 기능을 사용할 수 있습니다. 통합은 기본적으로 작동하며 웹 애플리케이션을 Dropbox 계정과 통합할 필요가 없습니다.

 	- 문서 템플릿에 서명 블록을 추가하여 비즈니스에서 자주 사용하는 견적서, 제안서, NDA 및 기타 문서를 생성하고 전자 서명을 위해 보낼 수 있습니다.

 	- 관리자는 자주 서명하는 사람을 수신자로 추가할 수 있으므로 영업 사원이 템플릿을 사용할 때마다 세부 정보를 입력할 필요가 없습니다.

***문서 템플릿에 서명 요소를 추가하는 방법***

 	
관리자 설정 &gt; 문서 템플릿 으로 이동합니다.

 	
새 템플릿을 만들거나 템플릿 목록에서 선택합니다.

![](/blog/images/post-6099-1759372074314-nu0806.png)

 	
문서 템플릿의 템플릿 설정에서 서명 요소를 템플릿으로 드래그 앤 드롭합니다.

오른쪽 창에서 서명 요소를 드래그 앤 드롭하여 별도의 섹션으로 추가할 수 있습니다.
표 안에 서명 요소를 원하는 경우 상단의 도구 모음에서 서명 자리 표시자를 사용하세요.

![](/blog/images/post-6099-1759372075353-iwcttz.png)

![](/blog/images/post-6099-1759372075785-pyqe8t.png)

 	
ADD SIGNATURE DETAILS 오버레이 에서  서명을 식별하기 위한 서명 레이블을 추가합니다.

연락처 모듈과 CRM의 사용자 목록에서 수신자를 선택할 수 있습니다.

현재 템플릿에서 사용한 기존 서명 중에서도 선택할 수 있습니다.

![](/blog/images/post-6099-1759372076036-c94ihk.png)

***문서 템플릿에서 서명 요소를 제거하는 방법***

 	
제거할 서명 요소를 클릭합니다.

 	
서명 설정 오버레이에서 서명 제거를 클릭합니다.

![](/blog/images/post-6099-1759372076317-gs7nba.png)

***문서 템플릿에서 서명 요소를 편집하는 방법***

 	
제거할 서명 요소를 클릭합니다.

 	
서명 설정 오버레이에서 서명 편집을 클릭합니다.

![](/blog/images/post-6099-1759372076581-xfp61i.png)

 	
서명 세부 정보 추가 오버레이 에서  서명자 세부 정보를 변경합니다.

![](/blog/images/post-6099-1759372076862-o4ljhr.png)

 	
서명 설정 창에서 서명 순서를 설정할 수 있습니다.

![](/blog/images/post-6099-1759372077111-whtkab.png)
