##### Freshworks x Freshchat x Freddy

**Freshworks의 채팅 솔루션 Freshchat 에서 새롭게 업데이트된 대화형 AI 챗봇 Freddy AI Agent 기능을 소개합니다!**

Freddy에서 나온 **AI Agent**는 URL, FAQ, 파일을 학습해서 자동으로 답변을 제공하는 모델로, **대화형 AI** 또는 **대화형 챗봇**입니다.
이 모델은 사용자가 자주 묻는 질문과 소스를 학습하여, 입력된 질문에 대해 적절한 답변을 자동으로 생성하는 기능을 수행합니다.

## Freddy AI Agent 안내

Freddy봇에 새롭게 출시된 기능인 AI Agent는 FAQ와 URL, 파일 소스를 빠른 시간 내에 자동 학습하고, 다국어 고객 문의를 번역하여 정확하게 답변합니다.

## AI Agent 기능 안내

### 1. 자동 답변

- 챗봇이 고객 질문을 이해하고 자동으로 답변합니다
- 답변의 출처를 URL, 문서, FAQ로 제공할 수 있습니다
- [Freddy AI 에이전트 소스 설정하기](https://crmsupport.freshworks.com/support/solutions/articles/50000010092-configure-knowledge-sources)

### 2. 실시간 언어 번역

- 다국어 언어 감지 및 자동 **유저↔에이전트 번역** 지원
- 학습 소스의 콘텐츠를 번역하여 답변합니다
- [AI Agent 실시간 다국어 지원](https://crmsupport.freshworks.com/support/solutions/articles/50000009979-auto-detect-customer-language-for-enhanced-bot-interaction)

Freddy AI 에이전트는 다국적 유저의 언어와 맥락을 이해합니다. 지속 학습을 통한 최적화된 답변을 제공하며, 상담원과 유저 언어 설정에 맞춰 실시간으로 대화를 번역합니다.

추가적으로, **AI Agent 이용은 Freshdesk Omni에서 추가 구매가 필요하지 않은 탑재된 기능입니다.**
별도 제품 구매 없이 채팅 이용을 위한 **봇 세션 추가**만으로 이용이 가능합니다.

## Freddy AI Agent 주요 장점

- **FAQ 학습**: 미리 정의된 질문과 답변을 학습하여 사용자 질문에 대한 자동 응답을 제공합니다
- **대화형 인터페이스**: 사람과의 대화를 통해 정보를 주고받는 형태로, 자연스러운 대화가 가능합니다
- **자동화된 고객 지원**: 고객의 문의에 신속하고 정확하게 응답하여, 시간과 자원을 절약할 수 있습니다
- **지속적인 학습**: 새로운 질문과 답변을 학습해 가며 점차적으로 더 나은 응답을 제공할 수 있습니다
