Tuesday, August 30, 2011**
페이지 번호와 페이지 수 기능 추가****구글문서도구의 문서는 그동안 페이지 넘버링 기능이 없었습니다.
초기에는 전체페이지를 통으로 보여주었기에 없었고 이후 페이징이 가능하게 개선이 되었지만 "3/15" 이런식으로 "전체 몇페이지 중에 몇페이지다"를 표시해주는 기능이 없었는데, 금번 구글독스 팀에서 페이지 넘버링 기능을 새로 추가했습니다.

이제 구글문서에서 페이지넘버와 카운트를 넣을 수 있게되었습니다. 사용법은 구글문서에서 삽입 > 페이지 번호**에서 가능하며 상단이나 하단을 선택할 수 있습니다.**

![](/blog/images/post-2328-1759372043310-vu4r2a.png)

추가적으로, 페이지 번호 이외 페이지 수도 추가되었는데 총 페이지 수를 나타냅니다. 페이지 번호와 페이지 수를 잘 조합하여 페이지 표시를 좀더 세부적으로 할 수 있습니다.

예를 들어, 페이지 머리글에 "Page 3 of 15"를 표시하고 싶다면 아래 과정을 따라해 보세요.

1. 메뉴에서 삽입 > 머리글****2. 우측 정렬 아이콘을 눌러 커서를 우측으로 이동시킵니다.
3. "Page"라고 타이핑 합니다.
4. 메뉴에서 삽입 > 페이지 번호 > 페이지 맨 위**로 이동하여 머리글에 페이지 번호를 삽입합니다.**5. "of"를 타이핑 합니다.
6. 메뉴에서 삽입 > 페이지 수**로 이동하여 총 페이지 수를 머리글에 삽입합니다.**

![](/blog/images/post-2328-1759372043352-f4w7kq.png)

원클릭 클라우드 프린팅**
올해 초에 구글은 [Google Cloud Print for mobile documents](http://googlemobile.blogspot.com/2011/01/cloud-printing-on-go.html)를 발표했었습니다. 베타서비스로 릴리즈된 구글 클라우드 프린트는 별도의 프로그램 설치 없이도 어떤 앱이나, device, OS, 브라우져에서 손쉽게 프린팅을 할 수 있게한 서비스입니다.

이번 업데이트에서는 모바일 구글문서도구에서 좀더 쉽게 원클릭으로 프린팅을 할 수 있도록 개선되었습니다.

모바일 폰에서 구글 문서도구앱으로 이동하여 출력을 원하는 문서를 선택 후 아래의 버튼 중 작업 > 인쇄를 누르면 클라우드 프린터에 설정된 프린터로 출력됩니다. 현재는 문서와 스프레드시트만 지원되고 있습니다.

![](/blog/images/post-2328-1759372043409-sq3df2.png)

구글 클라우드 프린트를 사용해보시기 위해서는 사용할 프린터를 구글 클라우드 [프린터에 연결](http://www.google.com/cloudprint)시켜야 합니다. 상세한 정보는 [구글 클라우드프린트 사이트](http://www.google.com/cloudprint/learn/)를 방문해 보시기 바랍니다.
